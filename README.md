## ☢️ 저장소 규칙

- 서로간의 충돌 방지를 위하여 각자 자신의 디렉토리를 생성하고 하위에서 정리합니다.
- 역시 서로간의 충돌 방지를 위해 자신의 브런치를 생성하여 작업하거나
  fork해서 PR 날려주세요.
- 작업후 커밋하시기 전에 다른 변경사항이 있는지 확인(fetch/pull) 해주세요
